### 1. What is the difference between null and undefined?

## Ans: 
- Undefined বিভিন্ন কারণে হতে পারে। যেমন কোনো ভ্যারিয়েবলের মান সেট না করলে সেখানে undefined দেখাবে। মানে ভ্যারিয়েবল ডিক্লেয়ার করা হয়েছে কিন্তু কোনো মান সেট করা হয়নি।

- null মানে ইচ্ছা করে খালি মান দেওয়া হয়েছে। মানে নিজেই ডিফল্টভাবে সেট করবো সেটি null দেখাবে।

### 2. What is the use of the map() function in JavaScript? How is it different from forEach()?

## Ans: 
- map() ব্যবহার করা হয় একটি array এর প্রতিটি উপাদানের উপর। এটি নতুন array return করে।

- forEach() ও array এর প্রতিটি উপাদানের উপর কিন্তু এটি কোনো নতুন array return করে না।

### 3. What is the difference between == and ===?

## Ans:
- ==  শুধু value তুলনা করে। প্রয়োজনে type পরিবর্তন করে।

- === value এবং type দুইটাই তুলনা করে। type পরিবর্তন করে না।

### 4. What is the significance of async/await in fetching API data?

## Ans:
- async/await ব্যবহার করলে asynchronous কাজ সহজভাবে করা যায় । API থেকে ডেটা আসতে সময় লাগে ।

### 5. Explain the concept of Scope in JavaScript (Global, Function, Block)?
## Ans: 

- Global Scope পুরো প্রোগ্রামে ব্যবহার করা যায় ।
- Function Scope শুধু সেই function এর ভিতরে ব্যবহার করা যায় ।
- Block Scope শুধু {} এর ভিতরে ব্যবহার করা যায় ।
